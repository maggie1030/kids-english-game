# 🌈 儿童英语游戏中心

一个专为儿童设计的英语学习网页游戏，包含单词配对和拼写挑战两个游戏模式。

**用途**: 自动部署测试项目，2026.03.02

## 🎮 游戏介绍

| 游戏 | 说明 | 适合年龄 |
|-----|------|---------|
| **单词配对** | 将图片与英文单词配对 | 3-6岁 |
| **拼写挑战** | 根据图片提示拼写单词 | 5-8岁 |

## 🚀 快速开始

### 本地游玩
直接双击 `英语游戏中心.html` 在浏览器中打开即可。

### 部署到腾讯云

#### 方式一：一键配置（推荐）
```bash
# 1. 运行配置脚本
./setup.sh

# 2. 编辑 deploy.sh，修改以下配置
BUCKET_NAME="你的存储桶名称"
REGION="你的地域（如 ap-hongkong）"

# 3. 执行部署
./deploy.sh
```

#### 方式二：GitHub Actions 自动部署
1. 创建 GitHub 仓库
2. 在 Settings → Secrets 添加腾讯云密钥
3. 推送到 main 分支，自动部署

详细步骤请参考 [自动化部署配置指南.md](自动化部署配置指南.md)
