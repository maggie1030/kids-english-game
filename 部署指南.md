# 🚀 腾讯云 COS 部署指南

## 方案选择

| 方案 | 适用场景 | 价格 | 复杂度 |
|-----|---------|-----|-------|
| **COS 静态网站** ⭐ | 纯静态页面（HTML/CSS/JS） | 极低（按流量） | ⭐ 最简单 |
| CVM 云服务器 | 需要后端服务 | 较高 | ⭐⭐⭐ 复杂 |
| 轻量应用服务器 | 小型网站 | 中等 | ⭐⭐ 中等 |

对于英语游戏，推荐 **COS 静态网站托管**。

---

## 📋 前置准备

### 1. 安装腾讯云 COS CLI 工具

```bash
# macOS
brew install tencent-cloud-cos-cli

# 或使用 pip
pip install coscmd

# 验证安装
coscli --version
```

### 2. 获取腾讯云密钥

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 进入「访问管理」→「API 密钥管理」
3. 创建密钥，获取：
   - `SecretId`
   - `SecretKey`

### 3. 创建 COS 存储桶

1. 进入 [COS 控制台](https://console.cloud.tencent.com/cos)
2. 点击「创建存储桶」
3. 配置：
   - 名称：`kids-english-game`（自定义，全局唯一）
   - 地域：选择离你最近的（如香港、上海）
   - 访问权限：**公有读私有写**
   - 开启「静态网站」功能

---

## 🔧 配置与部署

### 步骤 1：初始化配置

```bash
# 配置凭证（只需执行一次）
coscli config init

# 按提示输入：
# SecretId: 你的 SecretId
# SecretKey: 你的 SecretKey
# SessionToken: 直接回车（不需要）
# Endpoint: cos.ap-hongkong.myqcloud.com（根据你的地域修改）
```

**各地域 Endpoint 对照：**
- 香港：`cos.ap-hongkong.myqcloud.com`
- 上海：`cos.ap-shanghai.myqcloud.com`
- 北京：`cos.ap-beijing.myqcloud.com`
- 广州：`cos.ap-guangzhou.myqcloud.com`

### 步骤 2：修改部署脚本

编辑 `deploy.sh`，修改以下变量：

```bash
BUCKET_NAME="your-bucket-name"  # 你的存储桶名称
REGION="ap-hongkong"             # 你的存储桶地域
```

### 步骤 3：执行部署

```bash
# 给脚本执行权限
chmod +x deploy.sh

# 运行部署
./deploy.sh
```

## 🌐 访问网站

部署成功后，你可以通过以下地址访问：

```
https://your-bucket-name.cos-website.ap-hongkong.myqcloud.com
```

或者在 COS 控制台「静态网站」页面查看「访问节点」。

## 📝 自定义域名（可选）

1. 在 COS 控制台「域名管理」中添加自定义域名
2. 在域名服务商处添加 CNAME 记录
3. 配置 HTTPS 证书

## 🔄 后续更新

每次修改游戏后，重新运行：

```bash
./deploy.sh
```

脚本会自动：
- 删除旧文件
- 上传新文件
- 刷新 CDN（如果配置了）

## 💰 费用说明

COS 静态网站费用很低：
- **存储费用**：0.118元/GB/月（前 50GB 免费）
- **流量费用**：0.5元/GB（外网下行流量）
- **请求费用**：0.01元/万次

一个小游戏站点，月费用通常在 **1元以下**。

## ❓ 常见问题

### Q1: 部署失败，提示权限不足？
- 检查 SecretId/SecretKey 是否正确
- 检查存储桶访问权限是否为「公有读私有写」

### Q2: 访问网站显示 403？
- 确认存储桶已开启「静态网站」功能
- 确认默认首页设置为 `index.html`

### Q3: 如何绑定自己的域名？
- 在 COS 控制台「域名管理」中添加
- 需要备案（国内域名）

### Q4: 能否使用 HTTPS？
- COS 默认提供 HTTPS 访问
- 自定义域名可上传 SSL 证书
