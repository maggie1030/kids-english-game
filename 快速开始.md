# 🚀 快速开始 - 配置自动化部署

## 第一步：创建 GitHub 仓库（2分钟）

1. 打开 https://github.com/new
2. 仓库名填：`kids-english-game`
3. 选 **Public** ✅
4. 勾选 **Add a README**
5. 点击 **Create repository**

## 第二步：上传代码（2分钟）

```bash
cd ~/Desktop/儿童英语游戏
git init
git remote add origin https://github.com/你的用户名/kids-english-game.git
git add .
git commit -m "初始化项目"
git push -u origin master
```

## 第三步：添加密钥（1分钟）

1. 打开仓库页面 → 点击 **Settings**
2. 左侧 **Secrets and variables** → **Actions**
3. 点击 **New repository secret**，依次添加：

| Secret 名称 | 值（你的实际信息）|
|------------|----------------|
| `TENCENT_SECRET_ID` | AKIDxxxxxxxxxx |
| `TENCENT_SECRET_KEY` | xxxxxxxxxxxxxx |
| `TENCENT_BUCKET` | kids-english-game |
| `TENCENT_REGION` | ap-hongkong |

## 第四步：完成！🎉

1. 点击仓库的 **Actions** 标签
2. 等待 1-2 分钟，看到绿色 ✅
3. 访问：`https://你的存储桶.cos-website.地域.myqcloud.com`
